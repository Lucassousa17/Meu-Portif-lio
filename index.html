Sobre mim

Eu sou um Programador Front End, em formação, com grande entusiasmo por tecnologia e inovação. Atualmente, estou aprofundando meus conhecimentos em desenvolvimento web, com foco em HTML, CSS, JavaScript, Node.JS, React e Inteligência Artificial. Tenho experiência com projetos práticos onde pude aplicar conceitos de responsividade, APIs, integração com backend e boas práticas de programação.

Principais Experiências

HTML & CSS

Tenho conhecimento sólido em HTML e CSS, que são a base da construção de páginas web. Com HTML, sei estruturar conteúdos de forma semântica e acessível, usando corretamente as tags. Já com CSS, consigo criar estilos modernos e responsivos, trabalhando com propriedades como flexbox, grid, media queries e animações para deixar os sites bonitos e funcionais em diferentes dispositivos.

JavaScript

Tenho habilidades com JavaScript, uma linguagem essencial para tornar páginas web interativas e dinâmicas.Consigo manipular elementos da página (DOM), responder a eventos como cliques e envios de formulários, validar dados, consumir APIs externas e muito mais. Já trabalhei com funções, arrays, objetos, laços de repetição e lógica de programação.

React & Node JS

Tenho experiência com React, uma biblioteca JavaScript focada na criação de interfaces de usuário. Com React, desenvolvo componentes reutilizáveis que facilitam a construção de aplicações web modernas, rápidas e responsivas. Estou em constante aprendizado para dominar ainda mais essa poderosa ferramenta do front-end.
